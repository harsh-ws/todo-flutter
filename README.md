# Flutter Todo App

A clean and minimalist Todo application built with Flutter. This app helps users manage their daily tasks with a simple and intuitive interface.

## Features

- ✨ Create new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 💫 Clean and modern UI
- ⚡ Fast and responsive
